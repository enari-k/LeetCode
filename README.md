# LeetCode
## 概要
学生エンジニアenari_Kによる、LeetCodeの練習記録になります。ソフトウェアエンジニアリング協会のコード勉強会に参加しており、そこでコーディング試験対策を積むために、本リポジトリを使用しています。

## ディレクトリ構成
本リポジトリのディレクトリ構成は以下のようになっております。

```text
LeetCode
├── ProblemXXX                  # 問題XXX用のディレクトリです。
│   └── Solution.txt             # 各問題用のソリューションのファイルになります。 
└── README.md                   # 本ファイル、このリポジトリについて記されています。
```
